# 🏏 IPL 2026 Fan Website

A modern and responsive **IPL 2026 Fan Website** built using **HTML, CSS, and Bootstrap 5**.
This project showcases teams, matches, players, records, gallery, news, highlights, and more in an attractive UI.

---

## 🚀 Features

* 🔥 Hero Section with background image
* 🧢 Top Teams showcase
* 📅 Upcoming Matches table
* ⭐ Top Players section
* 📊 Points Table (Glass UI design)
* 🏆 IPL Records section
* 🖼️ Gallery with hover effects
* 📰 Latest News cards
* 🎥 Match Highlights (YouTube videos)
* 🧢 Orange Cap & Purple Cap section
* 📩 Contact Form
* 🔻 Stylish Footer with links & social icons

---

## 🛠️ Technologies Used
